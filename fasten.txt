:ID_PARAM_EXTRA_DEPTH  -  имя крепления


: - имя крепления начинается с двоеточия, после него
    идут поля крепления разделенные подчеркиванием "_"
    поэтому подчеркивание нельзя использовать внутри поля

ID - [строка] вид крепления, _желательно_ использовать международные обозначения,
     например М14 - для метрической резьбы 14мм,
     если таковых нет, то указывать размер крепления (мм - просто число, дюймы - ")

PARAM - [строка] опции крепления (папа, мама), варианты:
        f, female,  in, inner, internal - для мамы
        m,   male, out, outer, external - для папы
        u, uniform                      - для всего остального


EXTRA - [строка] анимация крепления, варианты:
        screw - резьба
        clamp - зажим

DEPTH - [число] глубина крепления, в тех же единицах, что и модель
        в направлении нормали крепления



детали соединяются двумя креплениями - по одному на каждой детали,
при этом крепления должны удовлетворять условиям:
	1. ID креплений должен совпадать
	2. комбинация PARAM креплений одна из следующих:
		(f + m), (m + f), (u + u)



примеры:

:M14_m_screw - болт
:M14_f_screw - гайка
:M14_u_clamp - шайба

:PIPE2"_in_clamp - двухдюймовая труба
:PIPE2"_out_clamp - фитинг на трубу
