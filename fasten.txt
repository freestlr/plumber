Крепление - объект в иерархии модели со специальным именем

Для правильного соединения деталей, объект-крепление нужно
расположить в точке по центру крепления, и повернуть таким образом,
чтобы ось X объекта указывала в сторону _от_ детали.


Две детали A и B, для соединения в креплениях A2 и B1,
будут расположены таким образом, что точки креплений
A2 и B1 будут совпадать, а их оси X будут коллинеарны и
направлены в противоположные стороны





:ID_PARAM_EXTRA_DEPTH  -  формат имени крепления


: - имя крепления начинается с двоеточия, после него
    идут поля крепления разделенные подчеркиванием "_"
    поэтому подчеркивание нельзя использовать внутри поля

ID - [строка] вид крепления, _желательно_ использовать международные обозначения,
     например М14 - для метрической резьбы 14мм,
     если таковых нет, то указывать размер крепления (мм - просто число, дюймы - ")

PARAM - [строка] опции крепления (папа, мама), варианты:
        f, female,  in, inner, internal,
        m,   male, out, outer, external,
        u, uniform,
        FP, MP


EXTRA - [строка] анимация крепления, варианты:
        screw - резьба
        clamp - зажим

DEPTH - [число] глубина крепления, в тех же единицах, что и модель
        в направлении нормали крепления



детали соединяются двумя креплениями - по одному на каждой детали,
при этом крепления должны удовлетворять условиям:
	1. ID креплений должен совпадать
	2. комбинация PARAM креплений одна из следующих:
		(f + m)
		(u + u)
		(FP + MP)
		(female + male)
		(uniform + uniform)
		(in + out)
		(inner + outer)
		(internal + external)



примеры:
	:M14_m_screw - болт
	:M14_f_screw - гайка
	:M14_u_clamp - шайба

	:PIPE2"_out_clamp - двухдюймовая труба
	:PIPE2"_in_clamp_2 - фитинг на трубу с выступом 2ед.
